### 부모자식선택자

* Tag를 이어붙이면 (EX: ul li) 부모-자식 관계로 CSS선택자 생성가능

* &#62;를 이용해서 직계 자식에게만 상속가능

* 쉼표(,)를 이용해서 여러 선택자 사용가능

### 가상 클래스 선택자
(Ex: 링크방문여부 )와 같이 동적인 활동을 가상 클래스로 하여 선택 가능
*    :link - 방문한 적이 없는 링크
*    :visited - 방문한 적이 있는 링크
*    :hover - 마우스를 롤오버 했을 때
*    :active - 마우스를 클릭했을 때  
(Ex: a:visied{})

___
### 부모 자식 선택자
 * border는 부모 자식 상속에서 하위 컴포넌트로 상속되지 않지만, color는 속성이 자식 컴포넌트로 상속됨.
 * \>는 바로 밑의 자식 속성이 일치하는 경우에만 설정이 적용됨
